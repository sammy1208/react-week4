## 排球位置用語

| 中文              | 英文縮寫 | 英文                         | 說明                                 |
| ----------------- | -------- | ---------------------------- | ------------------------------------ |
| 舉球員 / 二傳     | S        | Setter                       | 負責第二球組織攻擊、分配球路與節奏。 |
| 主攻手 / 邊攻     | OH / WS  | Outside Hitter / Wing Spiker | 多從左側攻擊，也常參與接發球與防守。 |
| 副攻手 / 中間手   | MB       | Middle Blocker               | 負責快攻與中央攔網。                 |
| 接應 / 反手位攻手 | OPP      | Opposite Hitter              | 常在右側攻擊，與舉球員對角站位。     |
| 自由球員          | L        | Libero                       | 後排防守專職，不在前排攻擊。         |
| 防守專員          | DS       | Defensive Specialist         | 後排防守替換角色。                   |
| 王牌              | Ace      | Ace                          | 隊伍主要得分攻手，不一定是位置名稱。 |
| 隊長              | C        | Captain                      | 場上精神核心與隊伍領導者。           |

## 排球動作用語

| 中文          | 英文                     | 日文常見                  | 說明                           |
| ------------- | ------------------------ | ------------------------- | ------------------------------ |
| 發球          | Serve                    | サーブ                    | 每回合開始的擊球。             |
| 跳發          | Jump Serve               | ジャンプサーブ            | 起跳後強力發球。               |
| 跳飄發        | Jump Float Serve         | ジャンプフローター        | 起跳發出的飄球，球路不穩定。   |
| 接發球 / 一傳 | Receive / Pass           | レシーブ                  | 接對方發球或攻擊，建立進攻。   |
| 舉球 / 托球   | Set / Toss               | トス                      | 通常第二球，由舉球員送給攻手。 |
| 扣球 / 攻擊   | Spike / Attack           | スパイク                  | 攻手將球打向對方場地。         |
| 快攻          | Quick Attack             | クイック                  | 舉球與攻擊節奏很快的攻擊。     |
| 變人速攻      | Freak Quick              | 変人速攻                  | 日向與影山的招牌速攻。         |
| 攔網          | Block                    | ブロック                  | 前排球員在網前阻擋攻擊。       |

| 吊球          | Tip / Feint              | フェイント                | 輕放球，避開攔網與防守。       |

## 《排球少年!!》常見作品內用語

| 用語         | 說明                                                       |
| ------------ | ---------------------------------------------------------- |
| 小巨人       | 日向憧憬的烏野小個子王牌形象，也是作品重要精神符號。       |
| 變人速攻     | 日向的速度與影山精準托球結合形成的超高速攻擊。             |
| 最強誘餌     | 日向後期被賦予的定位，靠存在感牽制對方攔網。               |
| 球場上的王者 | 影山國中時期的外號，帶有獨裁式舉球員的負面意味。           |
| 烏野的守護神 | 西谷夕的稱號，指自由球員守備能力。                         |
| 鐵壁         | 伊達工業的攔網代名詞。                                     |
| 垃圾場決戰   | 烏野與音駒的宿命對戰，烏鴉與貓的意象對決。                 |
| 全國前三王牌 | 指高中排球中頂級王牌群，常與牛島、佐久早、桐生等角色相關。 |
| 猜測攔網     | 天童覺的攔網風格，依直覺和觀察預判攻擊。                   |
| 同步攻擊     | 多名攻手同時助跑，讓對手難以判斷真正攻擊點。               |
